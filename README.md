# 🌍 Quiz App - Jeu de quiz sur la géographie 📍

🚀 **Version 1.0**  
🛠️ **Technologies** : React.js, Node.js, Express, MongoDB  

## 📌 **Description**
Quiz App est une application interactive qui permet de tester ses connaissances en géographie avec des questions sur :
- Les **capitales** des pays 🌍
- Les **drapeaux** des nations 🚩
- Les **monnaies** utilisées 💰

## ⚙️ **Installation et Démarrage**

### 1️⃣ **Cloner le projet**
```bash
git clone https://github.com/fefefefez/quiz-app.git
cd quiz-app
```

### 2️⃣ **Installer les dépendances**
📌 **Backend**
```bash
cd backend
npm install
```
📌 **Frontend**
```bash
cd ../frontend
npm install
```

### 3️⃣ **Démarrer l'application**
📌 **Lancer le Backend**
```bash
cd backend
npm run dev
```
📌 **Lancer le Frontend**
```bash
cd frontend
npm run dev
```

## 🚀 **Fonctionnalités**
✔️ Sélection de **catégories** de questions  
✔️ **15 questions** générées aléatoirement  
✔️ Affichage du **score final** après le quiz  
✔️ Option pour **rejouer** ou **changer de catégorie**  

## 🖥️ **Technologies utilisées**
- **Frontend** : React.js (Vite)
- **Backend** : Node.js (Express)
- **Base de données** : MongoDB  
- **Cache** (désactivé dans cette version) : Redis
